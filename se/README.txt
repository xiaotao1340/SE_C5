使用react搭建，主要使用ant design的组件。

除此之外还有很多安装好的组件，都在包里，所以应该不需要自己安装。

需要npm,node和react，以下指令均在CMD中即可。
检查有无:cmd中 node -v和npm-v
npm:npm install npm@latest -g
node:https://nodejs.org/
react:npm install react
顺带React DOM，可以渲染React元素，但应该不必要:npm install react-dom

运行需要cd到文件夹目录下，然后输入npm start
也可以直接用vscode打开文件夹，然后npm start
